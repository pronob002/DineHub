# DineHub

A Flutter-based mobile application developed to digitize university dining token management through secure authentication and a simple, student-friendly user experience.

## Overview

DineHub is a mobile solution designed to modernize the traditional dining token system used in university environments. Instead of relying on physical tokens or manually maintained records, the application provides a digital workflow that allows students to access, track, and manage dining-related activities more efficiently.

This project demonstrates practical experience in mobile application development, Firebase Authentication integration, and building software to solve a real-world institutional problem.

Traditional university dining systems often rely on physical tokens or manually tracked processes, which can be inefficient and inconvenient for students. Installation and Setup
Prerequisites
Before running the project locally, make sure you have:

Flutter SDK
Dart SDK
Android Studio or Visual Studio Code
A configured Firebase project
An Android device or emulator
Steps to Run Locally
Clone the repository:


git clone https://github.com/pronob002/DineHub.git
Move into the project directory:


cd DineHub/hall_dining
Install dependencies:


flutter pub get
Configure Firebase:
Create or open a Firebase project
Enable Email/Password Authentication
Add the required Firebase configuration file for your platform
For Android, place google-services.json in the appropriate directory
Run the application:


flutter run
